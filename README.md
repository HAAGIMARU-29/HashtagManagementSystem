# 📌 Hashtag Tracking and management System (Java OOP Project)

## 🚀 Overview

This project is a **Hashtag Management System** built using **Object-Oriented Programming (OOP) in Java**.
It simulates a basic social media-like environment where users can create posts, manage hashtags, and perform various operations.

---

## 🎯 Features

* 👤 User Management (Admin & Normal Users)
* 📝 Create and manage posts
* 🔍 Hashtag-based post filtering
* 📊 Report generation
* 📂 File handling for data storage (`users.txt`, `posts.txt`)
* 🧩 Modular OOP design using classes and interfaces
